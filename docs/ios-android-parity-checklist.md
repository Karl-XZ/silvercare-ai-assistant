# iOS / Android 功能对齐清单

本清单以 Android `MainActivity`、Java/Kotlin runtime 类和当前 iOS Swift 工程为对齐依据。状态含义：

- `done`：iOS 有对应功能，并有自动化测试或可编译验证。
- `partial`：iOS 有入口或 scaffold，但真实能力、下载、native runtime 或真机验证还不完整。
- `missing`：Android 有明确功能，iOS 还没有对应实现。
