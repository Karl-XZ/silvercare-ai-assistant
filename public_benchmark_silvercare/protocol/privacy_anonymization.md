# 脱敏规则

公开评测集必须满足以下规则。

## 图片和视频

- 不包含人脸。
- 不包含门牌、住址、可定位文本。
- 不包含证件、票据、个人身份材料。
- 重新编码图片，移除 EXIF 和设备元数据。
- 视频不保留原始设备元数据。
- 如果场景里出现可识别文本，必须裁剪、打码或替换样例。

## 语音

- 不包含真实姓名、联系方式、住址、账号、支付信息。
- 不包含可识别个人身份的连续对话。
- 文件名不含个人身份线索。

## Trace

- 只保留事件摘要，不保留原始传感器序列全文。
- 不记录个人身份、住址、照护机构名称。
- 时间使用相对毫秒，不使用真实日期时间。
- 可以保留任务结果、误报、人工修正和照护复核状态。

## 自动检查

运行：

```powershell
npm run benchmark:check
```

脚本会检查文本、JSONL、JPEG EXIF 标记、常见密钥模式和本机绝对路径。
